В файле learn.txt находятся данные для обучения.
В первом столбце находится номер дня существования сервиса. Во втором столбце находится размер аудитории сервиса на этот день.
Предлагается обучить предсказание размера аудитории в день t с помощью функции $N(t) = (\frac{1}{1 + e^{at+b}})N_0$

Решением должны быть значения a, b, N_0 и код, который их ищет.

Проверка качества решения будет осуществляться на последующих 365 днях, которых нет в обучающем датасете.

Код и решение следует присылать на адрес "saintnik@yandex-team.ru".
В теме письма должно быть написано "Домашняя работа 5 неделя Иванов И.И.", где вместо Иванов И.И. надо поставить свою фамилию и инициалы.
Код должен быть в прикреплён к письме в виде zip файла. Название файла должно быть вида Ivanov.zip. Вместо Ivanov в названии файла должна быть транслитерация фамилии студента.

Код необязательно должен состоять из одного файла. Главное и единственное требование к коду - будучи помещён в папку с исходными файлами датасета он должен выполнять задание. Никаких дополнительных ручных манипуляций с данными и кодом не предполагается при проверке выполнения задания.
